# Runpod SDK

There is a [SDK](https://docs.runpod.io/serverless/sdks) for Runpod serverless which is
[specified here](https://graphql-spec.runpod.io/).

As far as I can tell, the endpoint for this SDK is located at `https://api.runpod.io/graphql`.

This [hello world](hello-world) example tries to demonstrate the SDK.
